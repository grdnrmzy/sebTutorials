**E1.**

Variable - name to which data can be assigned.


| var    | example                  |
| -------- | -------------------------- |
| int    | playerHealth = 15;       |
| float  | playerHealth = 15f;      |
| bool   | playerHealth = true;     |
| string | playerHealth = 'health'; |

Functions means a specific task, so that it can be called from multiple places.
